EXCELLENCE FEATURES

COIN STRATEGY

30秒ごとにチャット最下部を自動確保。あなたのカジノパネルは常に「そこ」にある。

獲得した富で地位を築く。ショップ機能と連携し、サーバー内での特権をその手に。

HOW TO START

わずか3ステップで、あなたのサーバーに最高級のカジノを導入できます。

01

INVITE BOT

「Botを招待」ボタンからサーバーへ追加。必要な権限を許可してください。

02

SET CHANNEL

/setup コマンド（または指定のコマンド）をカジノ化したいチャンネルで実行します。

03

ENJOY CASINO

自動追尾パネルが出現。あとはメンバーと一緒にギャンブルを楽しむだけです。

MAIN COMMANDS

Command Action
/slot [bet] 5リールスロットを開始します
/coin [bet] [side] コインフリップ（表裏）に挑戦します
/balance 現在の所持金（草）を確認します
/panel 自動追尾パネルを手動で呼び出します
